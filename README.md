# homebrew-tap

## 通过 Homebrew 安装 rustfs

```sh
brew tap rustfs/homebrew-tap
brew install rustfs
```

## 支持平台

- macOS (x86_64, arm64)
- Linux (x86_64, arm64)

## 二进制包下载

可在 [GitHub Releases](https://github.com/rustfs/rustfs/releases) 页面下载各平台编译好的二进制包。

## 手动安装

下载对应平台的 zip 包，解压后将 rustfs 可执行文件放入 PATH 路径下即可。
